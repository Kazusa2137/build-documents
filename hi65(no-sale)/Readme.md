## 組み立てるのに必要な器具
| 名称 | 備考 |
|-----|-----|
| はんだごて | キースイッチやダイオードなどをはんだづけするためのもの。使用前に使用方法の動画などを閲覧することをおすすめします |
| はんだ線 | はんだずけするために必要不可欠のもの |
| ニッパー | ダイオードなどをカットするためのもの |
| ルブ | スタビライザーのかしゃかしゃ音を低減させるためのもの。なくても可 |
| ドライバー | 基盤のねじ止めに必要 |
